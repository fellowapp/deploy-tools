# Deploy Tools

Common deploy tools for Fellow infrastructure, provided as a Flox environment.

## Installed Packages

- **argocd** - Declarative GitOps CD tool for Kubernetes
- **fluxcd** (v0.35.0) - Continuous delivery solution for Kubernetes
- **jq** - Command-line JSON processor
- **jsonnet-bundler** - Package manager for Jsonnet
- **krane** - Kubernetes deployment tool by Shopify
- **kubectl** (v1.31) - Kubernetes command-line tool
- **kubernetes-helm** - Package manager for Kubernetes (Helm)
- **pulumi-bin** - Infrastructure as code platform
- **rage** - Simple, secure, and modern file encryption tool (Rust
  implementation of age)
- **sops** - Secrets OPerationS - encrypt/decrypt secrets files
- **stern** - Multi-pod and container log tailing for Kubernetes
- **tanka** - Flexible, reusable and concise configuration for Kubernetes using
  Jsonnet
- **yq-go** - YAML processor (Go version)
